git github Fundamentales

Git es un sistema de control de versiones distribuido que permite a los desarrolladores rastrear cambios en el código fuente a lo largo del tiempo. Es especialmente útil para proyectos colaborativos, ya que permite a múltiples desarrolladores trabajar en el mismo proyecto sin conflictos.

GitHub es una plataforma basada en la web que utiliza Git para el control de versiones. Proporciona una interfaz gráfica y herramientas adicionales para facilitar la colaboración entre desarrolladores, como la gestión de problemas, revisiones de código y despliegue continuo.
