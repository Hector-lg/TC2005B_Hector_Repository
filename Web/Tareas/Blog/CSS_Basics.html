¿Qué es CSS?

CSS (Cascading Style Sheets) es un lenguaje de estilos que se utiliza para describir la presentación de un documento HTML. Permite controlar el diseño, colores, fuentes y otros aspectos visuales de una página web.

Estructura Básica de CSS

selector { propiedad: valor; }

Componentes de CSS

  1. Selectores: Identifican los elementos HTML a los que se aplicará el estilo
  2. Propiedades: Características que se pueden modificar
  3. Valores: Configuraciones específicas para cada propiedad

Tipos de Selectores

Tipo de Selector Sintaxis Ejemplo
Selector de Tipo nombre_etiqueta { } p { color: blue; } (aplica a todos los párrafos)
Selector de Clase .nombre-clase { } .texto-importante { font-weight: bold; }
Selector de ID #nombre-id { } #encabezado { background-color: gray; }

Diferencias entre Selectores

  • Selector de Tipo: Afecta a todas las etiquetas del mismo nombre
  • Selector de Clase:
    • Puede aplicarse a múltiples elementos
    • Se usa con un punto (.) antes del nombre
    • Un elemento puede tener múltiples clases
  • Selector de ID:
    • Debe ser único en toda la página
    • Se usa con un # antes del nombre
    • Solo se puede aplicar a un elemento

Propiedades Comunes Utilizadas

  • color: Color del texto
  • background-color: Color de fondo
  • font-family: Tipo de letra
  • font-size: Tamaño de letra
  • margin: Espacio exterior
  • padding: Espacio interior
  • border: Bordes de los elementos
