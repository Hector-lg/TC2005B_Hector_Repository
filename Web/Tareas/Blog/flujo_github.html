Flujo de trabajo Git

Comandos escenciales

Comando Descripción
git clone URL Clona un repositorio remoto
git add . Agrega todos los cambios al área de preparación
git commit -m "mensaje" Guarda los cambios en el historial

Flujo de Trabajo

1. Clonar el Repositorio

Si aún no tienes el repositorio en tu computadora, usa el siguiente comando:

git clone https://github.com/usuario/repositorio.git

Esto descargará el código en una carpeta local.

2. Navegar al Repositorio

Accede a la carpeta del repositorio clonado:

cd repositorio

3. Crear o Editar Archivos

Modifica los archivos o agrega nuevos archivos a tu proyecto.

4. Agregar los Cambios al Área de Preparación

Para agregar un archivo específico:

git add archivo.html

Para agregar todos los cambios:

git add .

5. Confirmar los Cambios (Commit)

Registra los cambios en el historial de Git con un mensaje:

git commit -m "Descripción de los cambios"

6. Enviar los Cambios al Repositorio Remoto

Sube los cambios al repositorio remoto en la rama principal (usualmente main o master):

git push origin main

7. Verificar los Cambios

Visita el repositorio en GitHub para confirmar que los cambios han sido aplicados.
